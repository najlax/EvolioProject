Data Folder README
This folder is reserved for any datasets or data files required to run the Evolio project.
For this project, no external dataset files are required. The application uses a local SQLite database that is created and managed by the backend. The database can be initialized using the backend setup scripts included in the source code.
Data used by the project may include:
* User account information created through the application
* Uploaded project files stored through the backend upload system
* Database records generated while testing or running the application
* Seed data created using the provided backend scripts
Important Notes:
* No private, sensitive, or real user data is included in this submission.
* The actual database file should not be required for initial setup because it can be generated when running the backend.
* Any uploaded files created during testing should be treated as sample/testing data only.
* If a database file or sample uploads are included, they are only for demonstration purposes.
